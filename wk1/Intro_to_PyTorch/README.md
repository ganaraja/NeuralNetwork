# Summer Week 1 - AI Foundations

This repository contains the code and materials for Week 1 of the Summer AI Foundations course. The project is built using Python 3.12 and follows modern Python development practices.

## Project Structure

```
.
├── src/            # Source code directory
├── tests/          # Test files
├── docs/           # Documentation
├── .env/           # Environment variables and Private Keys
└── config.yaml     # Configuration file
```

## Prerequisites

- Python 3.12 or higher
- UV package manager


## Core Library Used
- **PyTorch**: Used for building and training neural networks, including custom activation functions


## Installation

1. Go to the root folder of this code repository:
   ```bash
   cd summer_week_1
   ```

2. Create and install a virtual environment:
   ```bash
   uv sync
   ```
